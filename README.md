# Multiple accounts


## Create keygen 
**ssh-keygen -t rsa -C "pritom.messi@gmail.com" -b 4096**

Add your own file name : id_myOwn


## Add created key to .ssh

**ssh-add ~/.ssh/id_myOwn**
